# 模板截圖指南

遊戲必須設定為 **1920x1080 解析度、繁體中文**。

## 如何截取

1. 打開 ZZZ AutoPlayer → 設定 → 工具 → 「截取模板截圖」
2. 或直接用 Windows 截圖工具 (Win+Shift+S)
3. 用圖片編輯器（如小畫家）裁切出下方指定的 UI 元素
4. 儲存為 PNG，放到對應資料夾

**重點：只裁切 UI 元素本身，不要包含太多背景。**

## 戰鬥模板 (`combat/`)

### 1. `dodge_signal.png` — 極限閃避提示

- **什麼時候出現**：敵人攻擊前，畫面會閃一下黃色/橘色光芒
- **在哪裡**：通常在角色周圍或畫面邊緣
- **截取什麼**：那個黃色閃光的提示圖示
- **怎麼觸發**：進入任意戰鬥，等敵人出手，在被打之前會看到

### 2. `ultimate_ready.png` — 大招可用提示

- **什麼時候出現**：終結技能量條滿的時候
- **在哪裡**：畫面右下角，角色頭像附近的能量條/圖示
- **截取什麼**：能量條滿時的發光狀態，或大招可用的按鈕提示
- **怎麼觸發**：戰鬥中持續輸出，等能量條集滿

### 3. `qte_prompt.png` — QTE 按鍵提示

- **什麼時候出現**：隊友觸發連攜時，畫面會顯示按鍵提示要你切人
- **在哪裡**：畫面中央或角色頭像旁
- **截取什麼**：那個「按空白鍵切人」的提示框
- **怎麼觸發**：戰鬥中用其他角色打出觸發條件

### 4. `switch_prompt.png` — 連攜攻擊提示

- **什麼時候出現**：可以觸發連攜攻擊時
- **在哪裡**：畫面中央偏下
- **截取什麼**：連攜攻擊的按鍵提示圖示
- **怎麼觸發**：戰鬥中完成特定連段後觸發

### 5. `combat_finished.png` — 戰鬥結束

- **什麼時候出現**：所有敵人被擊敗後
- **在哪裡**：畫面中央或上方
- **截取什麼**：「戰鬥結算」或「完成」的文字/畫面元素
- **怎麼觸發**：打完任意一場戰鬥

## 截取技巧

1. **盡量裁切小一點**：只包含辨識目標，不要截一大片背景
2. **避免截到會變動的數字**：例如不要把傷害數字截進去
3. **多截幾個版本**：如果同一個提示在不同場景長不一樣，選最常見的
4. **用遊戲內截圖**：不要用手機拍螢幕，要用截圖工具直接抓

## 範例尺寸參考

| 模板 | 建議尺寸 |
|------|---------|
| dodge_signal | 約 50x50 ~ 100x100 像素 |
| ultimate_ready | 約 60x60 ~ 80x80 像素 |
| qte_prompt | 約 100x60 ~ 150x80 像素 |
| switch_prompt | 約 100x60 ~ 150x80 像素 |
| combat_finished | 約 200x60 ~ 300x80 像素 |
